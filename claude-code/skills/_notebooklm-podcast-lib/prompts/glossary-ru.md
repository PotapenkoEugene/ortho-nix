## Common genetics terms
# Russian preferred → English abbreviation acceptable in parentheses
GWAS → полногеномный поиск ассоциаций (GWAS)
SNP → однонуклеотидный полиморфизм (SNP)
QTL → квантитативный локус признака (QTL)
LD → неравновесие по сцеплению (LD)
PCA → анализ главных компонент (PCA)
ADMIXTURE → программа ADMIXTURE
FST → индекс фиксации (Fst)
Ne → эффективный размер популяции (Ne)
SFS → спектр аллельных частот (SFS)

## Methods
PCR → ПЦР
WGS → секвенирование всего генома (WGS)
RNA-seq → транскриптомное секвенирование (RNA-seq)
ChIP-seq → ChIP-секвенирование
PSMC → метод PSMC (реконструкция демографии)
HapNe → метод HapNe (оценка Ne)
fastsimcoal2 → программа fastsimcoal2
BEAST → программа BEAST (байесовский анализ)

## Statistics
p-value → p-значение (произносить "п-значение")
FDR → уровень ложных открытий (FDR)
CI → доверительный интервал (ДИ)
OR → отношение шансов (ОШ)
HR → отношение рисков (ОР)
AUC → площадь под ROC-кривой (AUC)

## Common anti-anglicism rules
# Avoid: "significant" → use "значимый" or "достоверный"
# Avoid: "interesting" (интересно) as filler — it's meaningless
# Avoid: "basically", "essentially" → just say what it is
# Prefer: "авторы показали" over "результаты показывают"
# Prefer: "в данной работе" over "в данном исследовании" (when repetitive)
