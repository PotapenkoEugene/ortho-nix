Создай учебное руководство (study guide) по загруженным источникам для студентов биомедицинских специальностей.

Структура:
1. **Ключевые понятия** (5-10 терминов): термин → определение строго из источника + ссылка
2. **Центральный вопрос**: какую научную проблему решает эта работа?
3. **Дизайн исследования**: тип (RCT / observational / mechanistic / etc.), выборка, контроли
4. **Результаты для запоминания** (3-5 пунктов): самые важные находки с конкретными числами
5. **Распространённые заблуждения**: что студенты склонны неправильно понять про эту тему
6. **Вопросы для самопроверки** (5 штук): вопрос + правильный ответ из источника

ПРАВИЛА:
- Все определения — строго из текста источников, с пометкой [источник: N]
- Вопросы для самопроверки: ответ должен быть проверяем по тексту
- Если ответа нет в источниках — не включай вопрос
- Объём: 400-600 слов
