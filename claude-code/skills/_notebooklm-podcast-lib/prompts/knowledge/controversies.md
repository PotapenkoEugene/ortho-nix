Выяви противоречия, дискуссионные точки и нерешённые вопросы из загруженных источников.

Структура:
1. **Внутренние противоречия**: что источник утверждает в одном месте и ставит под сомнение в другом
2. **Методологические споры**: какие аспекты дизайна или интерпретации авторы сами считают спорными (limitations / discussion)
3. **Конфликт с другими источниками**: если загружено несколько документов — где они не согласны друг с другом (с указанием источников для каждой позиции)
4. **Открытые вопросы**: что явно остаётся нерешённым после этой работы
5. **Альтернативные интерпретации**: какие другие объяснения результатов возможны, если они упомянуты в источниках

Для каждого пункта:
```
**[Тема противоречия]**
Позиция А: [цитата или пересказ] [источник: N]
Позиция Б: [цитата или пересказ] [источник: M]
Статус: [не разрешено / частично разрешено / требует нового исследования]
```

ПРАВИЛА:
- Только противоречия, явно выводимые из загруженных источников
- Не выдумывай противоречия из фоновых знаний
- Если противоречий нет — честно напиши «источники не содержат явных противоречий»
