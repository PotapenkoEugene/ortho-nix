Составь хронологию (timeline) событий и этапов, описанных в загруженных источниках.

Формат: список событий в хронологическом порядке.

Включай:
- Даты предшествующих работ, упомянутых в источниках
- Этапы исследования (сбор данных, эксперименты, анализ)
- Публикации ключевых предшественников, если упомянуты
- Год публикации основной статьи и её место в общей хронологии поля

Для каждого пункта:
```
**[Год / период]** — событие или достижение
[Источник: N] — краткое обоснование включения
```

ПРАВИЛА:
- Только даты и события, явно упомянутые в загруженных источниках
- Если год не указан точно — пиши «~[год]» и отмечай неточность
- Не добавляй события из общих знаний — только из источников
- Если хронологии недостаточно для полного timeline — отметь это явно
