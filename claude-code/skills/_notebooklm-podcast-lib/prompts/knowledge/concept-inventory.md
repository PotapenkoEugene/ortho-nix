Составь инвентарь концепций (concept inventory) из загруженных источников — список ключевых идей с проверкой понимания.

Структура:
1. **Главная концепция**: центральная идея работы (1-2 предложения, строго из источника)
2. **Связанные концепции** (5-8 штук):
   - Название концепции
   - Как авторы её определяют или используют [источник: N]
   - Как она связана с главной концепцией
   - Типичное заблуждение о ней (если можно вывести из источника)

3. **Связи между концепциями**: опиши 3-5 причинно-следственных или корреляционных связей из источника

ПРАВИЛА:
- Заблуждения — только те, которые источник явно или косвенно опровергает
- Не добавляй концепции из фоновых знаний без ссылки на источник
- Укажи [источник: N] для каждого утверждения о концепции
