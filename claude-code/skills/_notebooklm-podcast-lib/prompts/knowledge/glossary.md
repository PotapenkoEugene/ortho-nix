Составь глоссарий терминов из загруженных источников.

Включай термины:
- Специфические методы (e.g., RNA-seq, ChIP-seq, GWAS, meta-analysis)
- Биологические понятия, определённые или использованные в тексте
- Статистические метрики (effect size, FDR, OR, HR и т.п.)
- Аббревиатуры, расшифрованные в источниках

Формат каждой записи:
```
**[Термин / аббревиатура]**
Определение: [строго из текста источника]
Контекст использования: [как авторы применяют этот термин]
[Источник: N]
```

ПРАВИЛА:
- Определение — только из текста, не из общих знаний
- Если термин используется, но не определён в источниках — пиши «определение в источниках не дано»
- Для генов/белков: указывай полное название при первом упоминании
- Объём: 10-20 терминов, приоритет — специфическим для этой работы
